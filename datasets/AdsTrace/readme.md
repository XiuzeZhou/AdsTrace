## 🛠️Dataset
1) Download the **AdsTrace** dataset from Hugging Face: https://huggingface.co/datasets/Xiuze/AdsTrace

2) Unzip the .zip files

3) Organize the data as follows:
```
├── datasets/AdsTrace/
|   ├── audios_16k/
│   ├── frames/
│   ├── ictr/
│   ├── transcripts/
│   ├── products_cn.json
│   ├── products_en.json
│   ├── tags_cn.csv
│   └── split.json
```
